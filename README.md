# rpi-deploy-site

Landing page for [rpi-deploy](https://github.com/khmilevoi/rpi-deploy),
served at https://rpi.iiskelo.com — deployed by rpi-deploy itself onto a
Raspberry Pi behind a Cloudflare Tunnel.

Status: bootstrap — spec and design approved, implementation not started.

Start here:

- `docs/HANDOFF.md` — **read this first**: context, hard facts, constraints,
  and the next step (writing the implementation plan).
- `docs/spec/2026-07-09-landing-page-design.md` — the approved design spec.
- `docs/spec/2026-07-09-landing-page-mockup.html` — the approved visual
  mockup (open in a browser; this is the design reference).
